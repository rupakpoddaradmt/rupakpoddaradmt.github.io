Aurex Privacy Policy

Last Updated: 11 March 2025

1. Information Collected:

Minimal User Information: Aurex requires an email address for login. The name and email address are collected and stored for authentication purposes.

2. Data Usage:

Functionality: The email address is used to authenticate users by requesting the server to send a One Time Password (OTP) to the provided email. This information is used only for login and is not shared with third parties.

3. Data Security:

Encryption and Secure Verification: All communication between the app and the server is encrypted using AES 256 encryption. During authentication, the app requests the server to send a One Time Password to the registered email address.

4. Changes to This Privacy Policy:

Updates: The Privacy Policy may be updated when necessary. Any changes will be reflected in the "Last Updated" date at the top. Checking this policy periodically is recommended.

5. Contact:

For any questions or concerns about this Privacy Policy, contact engineering@admtronics.com
