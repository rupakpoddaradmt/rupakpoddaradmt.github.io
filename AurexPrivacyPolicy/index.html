Minimal User Information: Aurex requires users to enter a pre-generated Serial Number for login purposes. No other personal information is collected or processed by the app.

Secure Verification: Aurex ensures data security by verifying the entered Serial Number with the server during the authentication process. All app functionality is encapsulated within the app itself, and no external data dependencies are required for its core features.
